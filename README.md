# Logger Project

Это проект для записи текстовых сообщений в журнал. В проекте реализованы две части:

- **Часть 1**: Библиотека для записи сообщений в журнал, которая записывает сообщения в текстовый файл.
- **Часть 2**: Консольное многопоточное приложение, которое использует библиотеку из первой части для записи сообщений в журнал.

### Часть 1: Библиотека записи сообщений

Библиотека предоставляет возможность записи сообщений в текстовый файл. При инициализации она принимает имя файла журнала и уровень важности сообщений. Запись осуществляется в файл с учётом уровня важности сообщений.

#### Особенности:
- Три уровня важности: низкий, средний, высокий.
- Возможность изменения уровня важности по умолчанию после инициализации.
- Сообщения с уровнем важности ниже установленного по умолчанию не записываются в журнал.
- Каждое сообщение содержит текст, уровень важности и временную метку.

### Часть 2: Консольное многопоточное приложение

Консольное приложение принимает сообщения от пользователя, передает их в отдельный поток для записи в журнал и затем снова ожидает ввода. Каждый поток записывает сообщения в журнал с учетом установленного уровня важности.

#### Особенности:
- Пользователь может вводить сообщения с уровнями важности или без.
- Приложение многозадачно: каждый ввод пользователя обрабатывается в отдельном потоке.
- Потокобезопасность гарантирована с использованием `std::mutex`.

## Сборка, запуск, очистка

Для сборки проекта используйте Makefile. Выполните команду:

```bash
make


Для запуска проекта. Выполните команду:

```bash
make run


Для очистки всех промежуточных файлов (объектных файлов и собранных приложений):

```bash
make clean
